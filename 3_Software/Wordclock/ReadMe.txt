Die Software wurde für das Board "ESP32 PICO-D4" programmiert.
Es wird die Board Library "Arduino ESP32 Boards" von Arduino in der Version 2.0.13 benötigt

Desweitern werden folgende Bibliotheken benötigt:
Adafruit NeoPixel by Adafruit Version V1.11.0
Wire by Arduino Version 2.0.0	(Standartmäßig installiert)
EEPROM by Arduino Version 2.0.0	(Standartmäßig installiert)
